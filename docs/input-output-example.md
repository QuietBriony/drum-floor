# Drum profile I/O example

## Input

- `vocal_intensity`: high
- `bass_density`: medium
- `guitar_chop`: high
- `section`: chorus
- `style_profile`: mixture_shout

## Output

- `kick`: follows bass accents
- `snare`: strong backbeat
- `hat`: tight 8th/16th hybrid
- `ghost_notes`: light and sparse
- `fill`: short pre-shout fill
- `crash`: section entry
- `transition`: lift into chorus peak
- `density`: high-mid
- `swing`: 6%
- `velocity_curve`: strong first beat, softened ghost layers
